Role User
1. user baru harus meregistrasikan diri terlebih dahulu
2. Setelah terdaftar user baru bisa menuju halaman login
3. Setelah login user bisa mencari buku yang ingin di baca di koleksi buku.
4. Terdapat 1 atau lebih buku yang disediakan oleh admin.
5. Jika status buku unavailable, user belum bisa meminjam buku (bermakna buku sedang dipinjam atau dalam maintenance)
